Hardware test section provides simple test of the phone functions.
#Camera - this folder contains camera view so you can see yourself,
you can select camera type (front/rear)
#Microphone - this folder contains page with microphone test service.
you can record your voice and then play it.
#Sound - this folder contains page with audio service.
You can play sound from Assert (Android) or Resources(iOS). 
Don't forget to set audio file properties to "Android Asset", "Bundle Resource" and "Do Not Copy";
